# 🌍 WanderLust

A **full-stack travel blogging platform** built to showcase modern web development, clean architecture, and production-ready practices.

> WanderLust allows users to discover, write, and share travel stories with authentication, caching, and a modern responsive UI.

---

## ✨ Features

- 🔐 Authentication & authorization
- 📝 Create, read, and manage travel blog posts
- ⚡ Redis caching for performance
- 🎨 Modern, responsive frontend UI
- 🧱 Clean backend architecture
- 🧪 Unit & integration testing
- 🐳 Dockerized frontend & backend
- 🔁 CI/CD ready (Jenkins)

---

## 🧩 Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Redis
- JWT Authentication
- Jest

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### DevOps
- Docker
- Docker Compose
- Jenkins
